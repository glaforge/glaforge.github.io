The article details the pervasive issue of "AI slop"—a term used to describe AI-generated text that, despite being grammatically correct, is often boring, repetitive, and devoid of a unique voice. This recognizable writing style, characterized by specific linguistic tics and structural crutches, has become ubiquitous due to the widespread adoption of large language models.

To combat this, the author introduces `tropes.fyi`, a website that meticulously tracks and lists these common patterns that betray AI authorship. Examples cited include the overuse of adverbs like "deeply" and "arguably," the ubiquitous "delve" family of verbs, and formulaic structural elements such as "It's not X, it's Y."

Motivated by a personal need to automatically refine his own AI-generated content, the author developed a solution: a custom skill named `deslopify` for the Gemini CLI.

The article explains the development process for `deslopify`. Leveraging Gemini CLI's built-in `skill-creator`, the author initiated the creation of a new skill, which automatically generated the necessary folder structure and a foundational `SKILL.md` file. Crucially, the author referenced the Markdown file provided by `tropes.fyi`, which contained a comprehensive list of AI sloppiness anti-patterns. The core of the `deslopify` skill lies within its `SKILL.md` file. This file contains a workflow instructing the AI agent to read the user's input text, cross-reference it against the anti-patterns from `tropes.fyi`, and then rewrite it. The explicit goal of this rewriting process is to make the text sound as if a "normal person wrote it," stripping away the artificiality. The article notes that the generated `SKILL.md` was so effective that no modifications were needed. The skill's simple structure comprises the `SKILL.md` file and a `references/` directory containing a `style_guide.md`.

For users interested in implementing `deslopify`, the article provides clear instructions. The code is hosted on GitHub, and Gemini CLI users can install it directly via a simple command: `gemini skills install https://github.com/glaforge/deslopify`, followed by reloading their interactive session. An important point highlighted is the portability of this skill format; because it's essentially a directory with a `SKILL.md` file and associated assets, it can potentially be used with any AI agent that supports this standard, with `agentskills.io` mentioned as a resource for understanding such structures.

Once installed, users can prompt their AI agent with commands like `"Deslopify this article: [URL]"`, `"Naturalize this draft: [paste text]"`, or `"Remove the AI slop from my README.md file."` The expected outcome is text that has been purged of "pompous phrasing and weird structural tics," resulting in significantly more readable content.

The author concludes by revealing a self-referential detail: a "good chunk" of the very blog post being summarized was initially generated using this `deslopify` skill within Gemini CLI. He clarifies that while the skill provided a valuable starting point, he meticulously reviewed and edited "each and every sentence," adding clarifications, paragraphs, screenshots, and links to refine the final output. This personal experience underscores the skill's utility in overcoming "blank page syndrome" and efficiently drafting project details, ultimately aiming to deliver content that avoids the very boredom it seeks to eliminate.
