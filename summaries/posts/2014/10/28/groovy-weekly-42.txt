**Key Releases and Updates:**
The week saw several important software releases. **Grails 2.4.4** was rolled out, bringing bug fixes and improvements to the popular web application framework. **Geb 0.10**, a browser automation solution, also saw a new release. A significant milestone was the release of **Reactor 2.0.0.M1**, the first milestone version which notably integrates with Reactive Streams, signaling advancements in reactive programming support for Groovy applications. Other releases included the **Groovy Android Gradle plugin 0.3.2**, which addressed joint compilation issues, and the **JMH Gradle plugin 0.1.3**. Furthermore, **Gretty 1.1.5**, a Gradle plugin for running web applications, was also made available.
